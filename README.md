# college Student Registration

Simple college registration form with PHP + MySQL.

## Features
- Unique College ID validation
- Glassmorphism design
- View students table
- Fully responsive

## Setup
1. Copy to `C:\xampp\htdocs\form\`
2. Create `veltech_db` database + students table
3. Start XAMPP (Apache + MySQL)
4. Visit: `localhost/form/form.html`
5. visit: `localhost/form/dashboard.php`
